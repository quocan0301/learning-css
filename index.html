12.Flexbox / Grid

a. Flexbox

Cấu trúc cơ bản của một phần tử flexbox với 3 flex item
  • Flex container: Chính là flexbox, nơi chứa các flex item bên trong nó.
  • Main axis: trục quyết định hướng mà các flex item sẽ hiển thị (mặc định từ trái qua phải); cross axis: trục còn lại của flexbox, vuông góc với main axis.
  • Cross start: vị trí bắt đầu của trục cross axis; cross end: vị trí kết thúc của trục cross axis.
  • Main start: vị trí bắt đầu của trục main axie; main end: vị trí kết thúc của trục ngang.
  • Flex item: các phần tử con chứa trong flexbox.
  • Main size: kích thước của flex item dựa theo hướng main axis.
  • Cross size: kích thước của flex item dựa theo hướng cross axis.

Để tạo ra một flex container, ta tiến hành khai báo:

      
        .container {
          display: flex;
        }

Một số attribute của flexbox

  • flex-direction: xác định hướng của main axis. 
                
              .container {
                display: flex;
                flex-direction: row; /* Hướng mặc định, từ trái qua phải */
                flex-direction: row-reverse; /* Từ phải qua trái */
                flex-direction: column; /* Từ trên xuống dưới */
                flex-direction: column-reverse; /* Từ dưới lên trên */
              }

  • flex-wrap: xác định flex item có được phép xuống dòng hay không. 
                
              .container {
                display: flex;
                flex-wrap: wrap; /* Mặc định, xuống dòng bình thường */
                flex-wrap: nowrap; /* Không xuống dòng, item sẽ được bọc trong container */
                flex-wrap: wrap-reverse; /* Xuống dòng nhưng chiều cross axis bị đảo ngược */

              }

  • flex-flow: là sự kết hợp của flex-direction và flex-wrap.
  • justify-content: xác định cách thức căn chỉnh các flex item theo trục ngang. 
                
              .container {
                display: flex;
                justify-content: flex-start; /* Mặc định, item hiển thị bắt đầu từ main start */
                justify-content: flex-end; /* Item hiển thị bắt đầu từ main end */
                justify-content: center; /* Item được căn giữa trục main axis */
                justify-content: space-between; /* Các item sẽ được căn chỉnh với khoảng cách giữa chúng bằng nhau 
                                                và hai item ở hai đầu sẽ được đẩy ra sát 2 bên */
                justify-content: space-around; /* Các item sẽ được căn chỉnh với khoảng cách giữa chúng bằng nhau 
                                                và khoảng cách đầu cuối item sẽ bằng 1/2 khoảng cách giữa các item */
                justify-content: space-evenly; /* Các item sẽ được căn chỉnh với khoảng cách giữa chúng bằng nhau 
                                                và khoảng cách đầu cuối item sẽ bằng khoảng cách giữa các item */
              }

  • align-items: xác định cách thức căn chỉnh các flex item theo chiều của cross axis. 
                
              .container {
                display: flex;
                align-items: flex-start; /* Mặc định, item hiển thị bắt đầu từ cross start */
                align-items: flex-end; /* Item hiển thị bắt đầu từ cross end */
                align-items: center; /* Item được căn giữa trục cross axis */
                align-items: stretch; /* Item sẽ được kéo dài theo chiều cross axis */
                align-items: baseline; /* Item sẽ được căn chỉnh theo baseline của chúng */
